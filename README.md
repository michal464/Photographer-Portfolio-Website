# Photographer-Portfolio-Website

Academic project developed as part of the HTML, CSS and JavaScript curriculum.

## Features
- Multi-page website
- Responsive design
- CSS Flexbox and Grid layouts
- Media Queries
- Forms and Navigation
- Images, Media and Iframes
- Modern UI design

## Technologies
- HTML5
- CSS3
- JavaScript
- Visual Studio Code

## Purpose
The project was created to demonstrate web development fundamentals, responsive design principles, and structured website architecture.
